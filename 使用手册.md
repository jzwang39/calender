# 仓库预约管理系统使用手册

## 1. 手册概述

本使用手册旨在帮助用户快速了解和使用仓库预约管理系统，包括系统功能、登录流程、各角色操作指南和常见问题解答等内容。

## 2. 系统简介

仓库预约管理系统是一个基于Web的预约管理平台，用于管理仓库的使用时段，实现仓库资源的高效利用。系统支持三种角色：仓库管理员、运营人员和客户，各角色具有不同的功能权限。

### 2.1 主要功能

- **用户认证**：登录、退出、修改密码
- **预约管理**：创建预约、查看预约、取消预约
- **时段管理**：设置关闭时段、查看关闭时段、取消关闭时段
- **用户管理**：查看用户列表、按角色筛选用户

## 3. 登录与退出

### 3.1 系统访问

打开浏览器，在地址栏输入系统访问地址：`http://localhost:3000`

### 3.2 登录系统

1. 在登录页面输入用户名和密码
2. 点击「登录」按钮
3. 系统验证成功后，根据用户角色自动跳转到相应的仪表盘

### 3.3 测试账号

|----------|----------|-----------|-----------------|

| operator | 123456   | 运营人员   | 管理客户和预约   |
| client1  | 123456   | 客户       | 创建和管理预约   |

### 3.4 退出系统

点击页面右上角的「退出登录」按钮，系统将清除用户会话并返回登录页面。

## 4. 角色功能指南

### 4.1 仓库管理员（admin）

#### 4.1.1 登录后页面

登录成功后，自动跳转到管理员仪表盘页面。

#### 4.1.2 主要功能

**1. 修改密码**
- 点击导航菜单中的「修改密码」
- 输入原密码和新密码
- 确认新密码后点击「保存」

**2. 管理预约**
- 查看所有客户的预约记录
- 可以按日期、客户等条件筛选

**3. 管理关闭时段**
- 点击「关闭时段管理」进入时段管理页面
- 设置关闭时段：
  - 选择日期和时段
  - 输入关闭原因
  - 点击「保存」
- 查看关闭时段列表
- 取消关闭时段：点击「取消」按钮

**4. 管理用户**
- 查看所有系统用户
- 按角色筛选用户

### 4.2 运营人员（operator）

#### 4.2.1 登录后页面

登录成功后，自动跳转到运营人员仪表盘页面。

#### 4.2.2 主要功能

**1. 修改密码**
- 点击导航菜单中的「修改密码」
- 输入原密码和新密码
- 确认新密码后点击「保存」

**2. 查看预约**
- 查看所有客户的预约记录
- 可以按日期、客户等条件筛选

**3. 管理客户**
- 查看所有客户信息
- 可以按客户名称或ID搜索

### 4.3 客户（client）

#### 4.3.1 登录后页面

登录成功后，自动跳转到预约页面。

#### 4.3.2 主要功能

**1. 修改密码**
- 点击导航菜单中的「修改密码」
- 输入原密码和新密码
- 确认新密码后点击「保存」

**2. 创建预约**
- 选择预约日期（只能选择工作日）
- 选择可预约的时段
- 填写预约信息（集装箱号、装箱单等）
- 点击「提交预约」

**3. 查看预约历史**
- 在预约页面下方查看所有预约记录
- 可以看到预约状态（活跃/已取消）

**4. 取消预约**
- 在预约历史列表中找到要取消的预约
- 点击「取消预约」按钮
- 确认取消操作

## 5. 常见操作流程

### 5.1 客户预约仓库流程

1. 客户登录系统
2. 进入预约页面
3. 选择预约日期（周一至周五）
4. 选择可预约的时段（未被他人预约且未被管理员关闭的时段）
5. 填写预约信息（集装箱号、装箱单等）
6. 点击「提交预约」
7. 系统显示预约成功信息

### 5.2 管理员设置关闭时段流程

1. 管理员登录系统
2. 进入关闭时段管理页面
3. 选择要关闭的日期
4. 选择要关闭的时段
5. 输入关闭原因
6. 点击「保存」
7. 系统显示关闭时段设置成功

### 5.3 运营人员查看预约流程

1. 运营人员登录系统
2. 进入仪表盘或预约管理页面
3. 查看所有客户的预约记录
4. 可以按日期、客户等条件筛选
5. 查看详细预约信息

## 6. 系统维护与故障排除

### 6.1 系统维护

- 定期备份数据库文件（database.db）
- 确保服务器正常运行
- 及时更新系统依赖包

### 6.2 常见问题与解决方案

**问题1：无法登录系统**
- 检查用户名和密码是否正确
- 确认网络连接是否正常
- 联系系统管理员重置密码

**问题2：无法选择某些日期或时段**
- 周六和周日不可预约
- 已被其他客户预约的时段不可选
- 被管理员设置为关闭的时段不可选

**问题3：预约提交失败**
- 检查填写的信息是否完整
- 确认所选时段是否可用
- 刷新页面重试

**问题4：忘记密码**
- 联系系统管理员重置密码
